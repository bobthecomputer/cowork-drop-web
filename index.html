GitHub

Repository and Pages deployment

1. Create a new empty GitHub repository. Name it something like cowork-drop-web.
2. Push this folder as its own repository. Commands are in the README below.
3. In GitHub, enable Pages with GitHub Actions. The workflow in this repo will build and publish automatically on every push to main.
